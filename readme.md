# Markov Assistant

## Supports
* Short-Term stock price's up & down prediction
* Long-Term stock price's movement
* Trading Bot Backtesting
* Trading Bot - Trade
* Simulating Trading Strategy with price, log rate return, SMA and VOL
* Stop Loss Monitoring
